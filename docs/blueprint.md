# **App Name**: Neptune Dive

## Core Features:

- Hero Banner with Call to Action: Visually engaging hero banner featuring an ocean background, tagline, and prominent call-to-action buttons ('Book a Dive,' 'Learn More').
- Course and Tour Display: Display list of courses with all required info.
- Image Gallery: Showcase a grid of diving photos with a lightbox preview to highlight underwater scenes.
- Team Profiles: Dedicated instructor team page that has the bios.
- Calendar and Booking System: An interface for checking availability.
- Contact and Location: An interface with map integration.

## Style Guidelines:

- Primary color: Deep ocean blue (#29ABE2), reflecting the depth and clarity of the underwater world.
- Background color: Desaturated light blue (#E1F5FE), creating a serene underwater atmosphere.
- Accent color: Coral orange (#FF7F50) for CTAs and interactive elements, providing a vibrant contrast.
- Body font: 'PT Sans', a humanist sans-serif providing modern look and readability.
- Headline font: 'Space Grotesk', a modern sans-serif for headlines and titles.
- Use clean, modern icons that represent diving equipment, marine life, and activities.
- Employ a mobile-first, responsive design to ensure the website is accessible and visually appealing on all devices.
- Implement smooth scroll-triggered animations for the hero and gallery sections to engage users as they explore the website.
